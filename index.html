Adding a recipe

Open data/recipes.json, copy one recipe object, and replace the values with a new title, description, category, tags, and the recipe URL.

If there is no URL, you can add the recipe details in the text field instead. 

{
  "title": "Your recipe title",
  "description": "A short description",
  "category": "Dinner",
  "time": "25 min",
  "tags": ["quick", "family"],
  "url": "https://example.com/your-recipe"
}

Ready for shared use

This project is designed for a small team or family to keep in one shared GitHub repository. Everyone can add recipes, and the site stays lightweight and mobile-friendly.

  • Open the repo on GitHub and edit the recipe data file.
  • Commit the change and publish it to GitHub Pages.
  • Anyone with the link can browse it from any device.
